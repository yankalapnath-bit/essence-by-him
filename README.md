# Essence by Him - Premium Perfume Website

Welcome to the official repository for **Essence by Him**, a luxury perfume e-commerce website built with pure HTML, CSS, and JavaScript.

## 🌟 Features

- **Elegant Design**: Premium, sophisticated layout matching the luxury perfume industry
- **Responsive Layout**: Fully responsive design that works on desktop, tablet, and mobile devices
- **Multiple Pages**: 
  - **Home**: Featured products and promotions
  - **Brand**: Our story and mission
  - **Perfume**: Complete product catalog with 8 unique fragrances
  - **Reviews**: Customer testimonials and ratings
  - **Contact**: Contact form and business information
- **Interactive Elements**: Hover effects, smooth navigation, form validation
- **Professional Styling**: Premium color scheme with gold accents

## 🚀 Getting Started

### Prerequisites
- A modern web browser (Chrome, Firefox, Safari, Edge)
- No build tools or dependencies required!

### Installation

1. Clone the repository:
```bash
git clone https://github.com/yankalapnath-bit/essence-by-him.git
cd essence-by-him
```

2. Open `index.html` in your web browser:
   - Double-click the file, OR
   - Right-click and select "Open with" → your browser

3. That's it! The website is ready to use.

## 🔧 Customization

To customize the website:

1. **Colors**: Edit color values in `styles.css`
2. **Products**: Edit product information in `perfume.html`
3. **Content**: Update text in any `.html` file
4. **Images**: Replace placeholder images with actual product photos
5. **Contact Info**: Update contact details in `contact.html`
